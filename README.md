# KeenKeeper

**Stay Close to the People Who Matter**

KeenKeeper is a personal relationship tracking app built to help you maintain meaningful connections in your life. It lets you keep track of your last interaction, set goals for staying in touch, and record different types of communication—all within a clean, responsive interface inspired by modern Figma-style design.


## 🌟 Core Features

- **Friends Dashboard** – Easily browse all your connections in a responsive grid layout. Each profile shows a picture, current status, and how long it has been since your last interaction.
- **Detailed Friend View** – Select any friend to access deeper insights, including interaction history, relationship goals, and a quick “Check-In” feature for logging calls, texts, or video chats.
- **Filtered Timeline** – Explore your complete interaction history and filter activities by type such as Call, Text, Video, or Meetup to quickly review past communication.
- **Interaction Analytics** – Gain insights into your communication patterns through a visual pie chart powered by Recharts.
- **Local Data Storage** – All your timeline data is securely stored in your browser using localStorage, so your history remains intact even after refreshing.
- **Responsive Experience** – Designed to work smoothly across mobile, tablet, and desktop devices.
- **Custom 404 Page** – Includes a simple and user-friendly error page for handling unknown routes.

## 🛠️ Tech Stack

- **React** – Frontend UI development
- **React Router DOM** – Navigation and routing
- **Context API + useReducer** – State management for timeline data
- **Tailwind CSS** – Utility-based styling
- **Recharts** – Chart and data visualization
- **React Toastify** – Notification system
- **React Icons** – Icon integration
- **Vite** – Fast build tool and dev server
- **LocalStorage** – Client-side data persistence
